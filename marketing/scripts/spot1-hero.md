# Spot 1 — Hero / Launch ("Care, finally on the same page.")

**Length:** 30s
**Word count target:** ~75 (150 wpm)
**Tone:** Calm, credible, operations-grade. Not corporate-warm.
**Voice direction:** Mid-tone neutral. Could be 35-50, any gender. Steady, unhurried. NOT salesy.
**Channels:** YouTube pre-roll, Vimeo embed on rayhealthevv.com, conference reel.

## Voiceover (verbatim — use this exact text for ElevenLabs)

Compliance proves itself. The result — care delivered, every shift, every time. RayHealth EVV. Care. Verified. Delivered.
